# Online_Retail_Dashboard

##  Project Overview

This project analyzes online retail sales data using PostgreSQL and Power BI. The goal was to uncover sales trends, identify top-performing products and countries, and generate business insights through interactive visualizations.

##  Tools & Technologies

* PostgreSQL (pgAdmin 4)
* Power BI
* Microsoft Excel
* DAX
* SQL

##  Project Workflow

1. Imported and explored the Online Retail dataset.
2. Performed data analysis using SQL queries.
3. Identified sales trends, customer behavior, and product performance.
4. Built an interactive Power BI dashboard.
5. Generated business insights from the analyzed data.

##  SQL Analysis Performed

* Total Revenue Analysis
* Top 10 Products by Revenue
* Top 10 Products by Quantity Sold
* Country-wise Order Analysis
* Average Revenue by Country
* Monthly Revenue Trend
* Top Countries by Quantity Sold
* Best Selling Products
* Revenue Ranking of Products
* Running Total Revenue Analysis
* Revenue Contribution by Country

##  Key Insights

* November generated the highest revenue.
* The United Kingdom was the top-performing country.
* A small number of products contributed significantly to total revenue.
* Sales performance varied across different months.
* Revenue contribution differed substantially between countries.
